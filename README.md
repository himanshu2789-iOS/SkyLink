
# SkyLink ✈️

A SwiftUI flight booking application that allows users to search flights, compare options, select seats, and review booking details.

## Features

- Flight search
- City picker
- Flight results listing
- Seat selection
- Booking summary
- Passenger details
- Modern SwiftUI UI components
- Reusable UI architecture

## Tech Stack

- Swift
- SwiftUI
- MVVM Architecture
- JSON Mock Data

## Project Structure

- Models
- Views
- ViewModels
- Services
- Resources
- ReusableViews
- Utilities

## Screens

- Flight Search
- Flight Results
- Seat Selection
- Passenger Details
- Booking Summary
